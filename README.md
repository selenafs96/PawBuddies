# PawBuddies

PawBuddies es una solución tecnológica integral diseñada para transformar
la gestión de las protectoras de animales en España, un sector que actualmente enfrenta graves ineficiencias debido al uso de herramientas analógicas y procesos dispersos. El proyecto consiste en una aplicación móvil multiplataforma (iOS y Android) desarrollada con React Native que centraliza la gestión de animales, voluntarios, procesos de adopción, donaciones e inventario. Basado en tres pilares fundamentales —innovación técnica, sostenibilidad estructural y digitalización avanzada—, el sistema elimina el uso de papel y optimiza la trazabilidad legal según la Ley 7/2023 de Bienestar Animal. A través de un diseño centrado en el usuario y una arquitectura en la nube con Supabase, PawBuddies no solo mejora la eficiencia operativa de las entidades del tercer sector, sino que también fomenta la cohesión social y la transparencia, facilitando que más animales encuentren un hogar responsable.

La aplicación ofrece módulos de: registro animal, gestión de turnos de voluntarios, pasarela de donaciones e inventario de suministros.
PawBuddies consiste en una solución tecnológica integral materializada en una aplicación móvil multiplataforma (iOS y Android) diseñada específicamente para profesionalizar la gestión de las protectoras de animales. El producto se ha desarrollado utilizando el framework React Native y JavaScript, lo que permite un despliegue eficiente en ambos sistemas operativos con un único código base, optimizando así los recursos de desarrollo y mantenimiento.

La solución final centraliza en un único entorno digital procesos que tradicionalmente se gestionan de forma dispersa:
- Gestión integral de animales: Incluye fichas digitales detalladas con historiales médicos, seguimiento sanitario y trazabilidad completa de cada ejemplar.
- Módulo de adopciones: Digitaliza las solicitudes de adopción.
- Voluntariado: permite registro de voluntarios y que éstos puedan gestionar la información de la protectora desde la aplicación.
- Sección de noticias: Funciona como un canal de comunicación para difundir eventos, animales perdidos y campañas de sensibilización.

El servicio está orientado principalmente a tres segmentos interconectados:
- Protectoras de animales de tamaño pequeño y mediano: Que carecen de recursos para software a medida y operan con métodos obsoletos.
- Voluntarios y gestores: Usuarios críticos que necesitan reducir su carga administrativa diaria y acceder a la información en tiempo real desde cualquier ubicación.
- Ciudadanía (adoptantes): Personas interesadas en adoptar o colaborar, que buscan un catálogo actualizado y un proceso de interacción fluido y transparente.
